5. Iterable, Iterator, Generator

5.1 Iterable

  • Iterable의 정의:
    • __iter__ 또는 __getitem__을 구현한 객체.
    • for 문과 iter() 함수와의 관계.
  • 실습: 커스텀 Iterable 클래스 구현.
  • Iterables Docs

5.2 Iterator

  • Iterator의 정의:
    • __iter__와 __next__ 메서드를 구현한 객체.
    • StopIteration 예외 처리와 반복 종료.
  • 실습: 커스텀 Iterator 클래스 작성.

5.3 Generator

  • Generator의 정의:
    • yield 키워드로 구현.
    • 상태를 유지하며 값을 생성하는 함수.
  • Generator Expressions:
    • 리스트 내포와 유사한 문법.
    • 차이점: 메모리 효율성과 즉시 실행.
